Instrucciones para actualizar productos:

  1. Completa todos los campos del formulario con la información del nuevo producto.
  2. Para las imágenes, proporciona enlaces directos a imágenes de alta calidad (recomendamos Unsplash o servicios similares).
  3. El "Badge" es opcional y se usa para mostrar etiquetas como "Nuevo", "Más Vendido", etc.
  4. Haz clic en "Generar Código" para ver una vista previa del producto y obtener el código HTML.
  5. Copia el código generado y pégalo en la sección correspondiente de la landing page.
